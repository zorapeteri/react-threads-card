  <tbody>
    <tr>

      <td></td>

    </tr>
    <tr>
      <td>&nbsp;&nbsp;- username</td>
      <td>Threads handle of the author</td>
      <td>string</td>
      <td></td>
      <td>Required</td>
    </tr>
    <tr>
      <td>&nbsp;&nbsp;- image</td>
      <td>URL to the profile picture of the author</td>
      <td>string</td>
      <td></td>
      <td>Required</td>
    </tr>
    <tr>
      <td>&nbsp;&nbsp;- verified</td>
      <td>Indicates whether the author is verified</td>
      <td>boolean</td>
      <td>false</td>
      <td>Optional</td>
    </tr>
    <tr>
      <td>text</td>
      <td>The text of the post itself</td>
      <td>string</td>
      <td></td>
      <td>Required</td>
    </tr>
    <tr>
      <td>time</td>
      <td>
        Time of posting. If passed in as a Date object, it will be formatted as

      </td>
      <td>Date | string</td>
      <td></td>
      <td>Required</td>
    </tr>
    <tr>
      <td>permalink</td>
      <td>
        The URL for the post (or anything, it's up to you). Clicking the Threads

      </td>
      <td>string</td>
      <td></td>

    <tr>

        <a href="/intro#adding-images">images</a>

      <td></td>
      <td>Optional</td>
    </tr>
    <tr>
      <td>&nbsp;&nbsp;- src</td>

      <td>string</td>
      <td></td>
      <td>Required</td>
    </tr>
    <tr>
      <td>
        <a href="/intro#displaying-engagement">engagement</a>
      </td>
      <td>
        The number of engagements (replies, reposts, likes) the post has

      <td></td>
      <td>Optional</td>
    </tr>
    <tr>

      <td>Number of replies</td>
      <td>number</td>

      <td>Optional</td>

      <td>&nbsp;&nbsp;- reposts</td>

      <td>0</td>

    </tr>
    <tr>

      <td>number</td>
      <td>0</td>
      <td>Optional</td>
    </tr>
    <tr>

      <td>boolean</td>

      <td>Optional</td>

      <td>Optional</td>

      <td>showEngagement</td>

      <td>true</td>
      <td>Optional</td>

Try them out on this playground:

```jsx live
<ThreadsCard
  className="with-background-image"
  author={{
    username: 'ketchup.jessi',
    image: 'img/ketchup.jessi.jpg',
    verified: false,
  }}
  text="how do 21 pilots even fit in the cockpit fr"
  time={new Date('2024-11-09T21:13:00Z')} // try a string instead!
  permalink="https://www.threads.net/@ketchup.jessi/post/DCKk_pDpwhA"
  engagement={{
    likes: 1300,
    reposts: 86,
    replies: 26,
  }}
  theme="light"
  colors={{}}
  gradientBackground={false}
  blurredBackground={false}
  clickableProfileLink={false}
  showEngagement={true}
  showTime={true}
/>
```
